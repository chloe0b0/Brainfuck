### Usage
From a terminal
```
Brainfuck.exe -i ++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.++++++..+++.>>.<-.<.+++.------.--------.>>+.>++.

>>> Hello World!
```

Interpret files
```
Brainfuck.exe -f HelloWorld.bf

>>> HelloWorld!
```
